# AI-Based Productivity Analyzer

## Overview
The AI-Based Productivity Analyzer is a web-based application that evaluates a user's daily routine and calculates a productivity score. Based on the score, the system classifies the day as Productive, Moderate, or Lazy and provides suggestions for improvement.

---

## Features
- Rule-based AI-inspired productivity analysis  
- Real-time score calculation (0–100)  
- Productivity classification (Productive / Moderate / Lazy)  
- Personalized suggestions  
- Dark-themed user interface  
- Fully browser-based (no backend required)  

---

## Problem Statement
Many individuals track their daily activities but lack a structured way to evaluate productivity. This project provides a simple solution by analyzing daily habits and converting them into a measurable productivity score.

---

## How It Works
The user inputs:
- Study/Work Hours  
- Exercise Hours  
- Sleep Hours  
- Screen Time  
- Goals Completed  

The system processes this data using predefined rules to generate a score.

---

## Scoring Logic
- More study/work hours increase the score  
- Exercise adds positive value  
- 7–8 hours of sleep improves productivity  
- Completing more goals increases the score  
- High screen time reduces the score  

---

## Productivity Classification

| Score Range | Category   |
|------------|-----------|
| 70 – 100   | Productive|
| 40 – 69    | Moderate  |
| 0 – 39     | Lazy      |

---

## Tech Stack
- HTML  
- CSS  
- JavaScript

## Setup Instructions
1. Create a folder named `AI-Based Productivity Analyzer`  
2. Add the following files:
   - `index.html`  
   - `style.css`  
   - `script.js`  
   - `README.md`  
3. Copy the respective code into each file
