## Introduction  

This repository contains my **AI-Based Productivity Analyzer**, developed as part of the **Bring Your Own Project (BYOP)**.  

The objective of this project is to analyze a user’s daily activities and determine whether the day was **Productive**, **Moderate**, or **Lazy**. The system helps users reflect on their daily routines and encourages better time management and productivity habits.  

The project uses an **AI-inspired rule-based approach** to simulate intelligent decision-making and provide meaningful feedback.

## Problem Statement  

In today’s fast-paced digital environment, students often struggle to evaluate how effectively they utilize their time. Without proper reflection, it becomes difficult to improve productivity and maintain discipline.  

This project addresses this issue by providing a simple and interactive system that analyzes daily activities and offers insights into productivity levels.

## System Overview  

The application allows users to input a description of their daily activities. Based on this input, the system evaluates productivity using predefined rules and generates output in the form of classification, score, and suggestions.

---

## Methodology  

The system follows a rule-based approach that simulates basic Artificial Intelligence:

1. **Input Processing**  
   The user enters daily activities in text form.

2. **Keyword Analysis**  
   The system scans for keywords associated with productive and unproductive activities.

3. **Scoring Mechanism**  
   - Productive activities increase the score  
   - Unproductive activities decrease the score  

4. **Classification**  
   Based on the final score:  
   - High score → Productive Day  
   - Medium score → Moderate Day  
   - Low score → Lazy Day  

5. **Feedback Generation**  
   The system provides suggestions to improve productivity.
